# RGB PROJECT (SIMPLE) 
Using an Arduino Uno to control a LED RGB wire, is necesary:
- Arduino UNO
- Transistor 2N2222
- Resistor 220 Ohms
- 12V Power Supply
- LED RGB WIRE [12V,R,G,B]
